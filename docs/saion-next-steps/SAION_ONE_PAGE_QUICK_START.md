# SAION One-Page Quick Start

Use this quick guide when you want to move fast.

## 1. Start

1. Open SAION.
2. Set a session title.
3. Choose your mood/style direction.

## 2. Create Your First Version

1. Enter your prompt.
2. Adjust core controls (emotion, vocal feel, timing, effects).
3. Choose a generator.
4. Click Generate.

## 3. Listen and Improve

1. Play the new result.
2. Adjust one or two controls only.
3. Generate again.
4. Repeat until the result feels right.

## 4. Use A/B Compare

1. Save a good result as Version A.
2. Create a variation and save as Version B.
3. Switch between A and B.
4. Keep the stronger version.

## 5. Optional Visual Personalization

1. Open visual controls.
2. Pick a look that fits your workflow.
3. Optionally add a texture source URL.
4. Confirm update success message.

## 6. Save and Export

1. Save your session.
2. Export notation or session snapshot.
3. Use clear names for versions.

## 7. If Something Feels Off

1. Retry once.
2. Simplify the prompt.
3. Reduce how many controls you change at once.
4. Return to last good version and continue.

## Daily Fast Flow

1. Set mood
2. Generate baseline
3. Save A
4. Refine and generate
5. Save B
6. Compare
7. Export best
